Ejercicio 1
Café de Barrio
CF=$1.200 · PVU=$5 · CVU=$2
¿Cuántos cafés para cubrir costos?
PE = 400 unidades
MCU = $5.00 − $2.00 = $3.00
PE = $1.200 ÷ $3.00 = 400 cafés
Ingresos PE = 400 × $5 = $2.000
Ejercicio 2
Taller de Bicicletas
CF=$1.800 · PVU=$8 · CVU=$3,50
¿Cuántas reparaciones al mes?
PE = 400 reparaciones
MCU = $8.00 − $3.50 = $4.50
PE = $1.800 ÷ $4.50 = 400 unidades
Ingresos PE = 400 × $8 = $3.200
Ejercicio 3
CF Sube — Taller
CF=$2.400 · PVU=$8 · CVU=$3,50
¿Cómo cambia el PE al subir el arriendo?
PE sube a 534 unidades (+134)
MCU = $8.00 − $3.50 = $4.50 (igual)
PE = $2.400 ÷ $4.50 = 534 unidades
Conclusión: más CF → necesitas vender más para equilibrar.
Ejercicio 4
Opción A: Bajar Precio
CF=$1.800 · PVU=$7 · CVU=$3,50
¿Conviene reducir el precio de $8 a $7?
PE sube a 514 unidades ❌ Peor
MCU = $7.00 − $3.50 = $3.50 (bajó de $4.50)
PE = $1.800 ÷ $3.50 = 514 unidades
Bajar PVU reduce el MCU → necesitas vender MÁS.
Ejercicio 5
Opción B: Bajar CVU
CF=$1.800 · PVU=$8 · CVU=$3,00
¿Conviene negociar insumos más baratos?
PE baja a 360 unidades ✅ Mejor
MCU = $8.00 − $3.00 = $5.00 (subió de $4.50)
PE = $1.800 ÷ $5.00 = 360 unidades
Bajar CVU aumenta MCU → necesitas vender MENOS para cubrir costos.

Inelástica |ε| < 1 → Al subir precio, IT aumenta
Fórmula:
ε = ΔQ% ÷ ΔP%
Elástica |ε|>1: bajar P → sube IT
Inelástica |ε|<1: subir P → sube IT
Unitaria |ε|=1: IT no cambia

Ejercicio 6
Zapatillas Deportivas
P1=$50 · Q1=500 · ε=0.8 · P2=$55
¿Subir el precio sube los ingresos?
SÍ — Inelástica ✅
ΔP% = +10% → ΔQ% = −0.8×10% = −8%
Q2 = 500×0.92 = 460 unidades
IT1 = $50×500 = $25.000
IT2 = $55×460 = $25.300 (+$300)
|ε|=0.8 < 1 → subir P sube IT ✓
Ejercicio 7
Restaurante Comida Rápida
P1=$10 · Q1=400 · ε=1.5 · P2=$9
¿Bajar el precio sube los ingresos?
SÍ — Elástica ✅
ΔP% = −10% → ΔQ% = −1.5×(−10%) = +15%
Q2 = 400×1.15 = 460 unidades
IT1 = $10×400 = $4.000
IT2 = $9×460 = $4.140 (+$140)
|ε|=1.5 > 1 → bajar P sube IT ✓
Ejercicio 8
Notebook / Laptop
P1=$80 · Q1=200 · ε=2.1 · P2=$72
Demanda muy elástica — ¿bajar precio?
SÍ — Muy Elástica ✅
ΔP% = −10% → ΔQ% = −2.1×(−10%) = +21%
Q2 = 200×1.21 = 242 unidades
IT1 = $80×200 = $16.000
IT2 = $72×242 = $17.424 (+$1.424)
|ε|=2.1 > 1 → demanda responde con fuerza ✓
Ejercicio 9
Medicamento Básico
P1=$30 · Q1=600 · ε=0.2 · P2=$33
Casi inelástico — ¿qué pasa con Q?
Q baja muy poco — Muy Inelástica
ΔP% = +10% → ΔQ% = −0.2×10% = −2%
Q2 = 600×0.98 = 588 unidades
IT1 = $30×600 = $18.000
IT2 = $33×588 = $19.404 (+$1.404)
|ε|=0.2 → la gente compra igual aunque suba el precio ✓
Ejercicio 10
Elasticidad Unitaria
P1=$40 · Q1=300 · ε=1.0 · P2=$44
¿Cambian los ingresos totales?
Ingresos casi iguales — Unitaria
ΔP% = +10% → ΔQ% = −1.0×10% = −10%
Q2 = 300×0.90 = 270 unidades
IT1 = $40×300 = $12.000
IT2 = $44×270 = $11.880 (≈ igual)
|ε|=1.0 → el alza de P se cancela con la caída de Q ✓
