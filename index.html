Ejercicio 1
Panadería "El Buen Pan"
CF=$5,000 · PVU=$2.00 · CVU=$0.80 → ¿Cuántos panes para cubrir costos?
PE = 4,167 panes
MCU = $2.00 − $0.80 = $1.20
PE = $5,000 ÷ $1.20 = 4,167 unidades
Ingresos PE = 4,167 × $2.00 = $8,334
Ejercicio 2
Mesas de Oficina
CF=$24,000 · PVU=$200 · CVU=$80 → ¿Cuántas mesas vender?
PE = 200 mesas
MCU = $200 − $80 = $120
PE = $24,000 ÷ $120 = 200 unidades
Ingresos PE = 200 × $200 = $40,000
Ejercicio 3
Costos Fijos Suben
CF=$30,000 · PVU=$200 · CVU=$80 → ¿Cómo cambia el PE?
PE = 250 mesas (sube en 50)
CF aumenta de $24,000 a $30,000 (+25%)
PE = $30,000 ÷ $120 = 250 unidades
Conclusión: Más CF → más unidades para cubrir costos.
Ejercicio 4
Opción A: Bajar Precio
CF=$24,000 · PVU=$180 · CVU=$80 → ¿Conviene bajar el precio?
PE sube a 240 mesas ❌ Peor
MCU = $180 − $80 = $100
PE = $24,000 ÷ $100 = 240 unidades
Bajar precio reduce MCU → necesitas vender MÁS para equilibrar.
Ejercicio 5
Opción B: Bajar CVU
CF=$24,000 · PVU=$200 · CVU=$70 → ¿Cuánto mejora el PE?
PE baja a 185 mesas ✅ Mejor
MCU = $200 − $70 = $130
PE = $24,000 ÷ $130 = 184.6 ≈ 185 unidades
Reducir CVU aumenta MCU → necesitas vender MENOS para equilibrar.

📊
Inelástica |ε| < 1 → Al subir precio, los ingresos AUMENTAN
Fórmula de Elasticidad:
ε = (ΔQ% / ΔP%) = [(Q2/Q1)−1] / [(P2/P1)−1]
Regla clave: si |ε| > 1 = elástica (bajar precio sube IT). Si |ε| < 1 = inelástica (subir precio sube IT).
Clasificación Rápida
Tipo |ε| Ejemplo
Inelástica < 1 Leche, medicamentos
Unitaria = 1 Transporte básico
Elástica > 1 Smartphones, joyas

Ejercicio 6
Zapatillas Deportivas
P1=$50, Q1=10.000, ε=−0.8, P2=$55 → ¿Debería subir el precio?
SÍ debería subir el precio ✅
ΔP% = +10% → ΔQ% = −0.8 × 10% = −8%
Q2 = 9,200 unidades
IT1 = $500,000 → IT2 = $506,000 (+$6,000)
|ε| = 0.8 < 1 = Inelástica → subir P sube IT
Ejercicio 7
Restaurante Comida Rápida

SÍ debería bajar el precio ✅
ΔP% = −10% → ΔQ% = −1.5 × (−10%) = +15%
Q2 = 575 platos
IT1 = $5,000 → IT2 = $5,175 (+$175)
|ε| = 1.5 > 1 = Elástica → bajar P sube IT
Ejercicio 8

Ejercicio 9
Medicamento Esencial

Ejercicio 10
Elasticidad Unitaria
P1=$100, Q1=3.000, ε=−1.0, P2=$110 → ¿Cambian los ingresos?
Ingresos casi iguales — elasticidad unitaria
ΔP% = +10% → ΔQ% = −10%
Q2 = 2,700 unidades
IT1 = $300,000 → IT2 = $297,000 (casi igual)
|ε| = 1.0 = Unitaria → el cambio en P se compensa con el cambio en Q
